Ini merupakan repositori Tugas Pendahuluan modul 5 Praktikum Pemecahan Masalah dengan Pemograman milik praktikan Muhammad Aqeel Ghani (NIM. 13224071). Semoga repositori ini dapat dipergunakan dengan baik untuk membantu Asisten menilai praktikan. Naskah Soal Tugas Pendahuluan dapat diakses pada file `Modul 5 - Naskah Soal Tugas Pendahuluan EL2008 PMP.pdf`

## Soal Teori

Untuk soal teori, jawabannya berada pada file `Tugas Pendahuluan Modul 5.pdf`. Selain itu terdapat source code untuk `soal 1.b.`pada file `ContohDFS.c` dan `ContohBFS.c`.

## Soal Pemograman

Source code soal pemograman berada pada file `SoalPemograman.c`.

### Contoh Test Case

#### Test Case 1 (Input Valid)
```
Masukkan orang pertama: Omen
Masukkan orang kedua: Breach
Leluhur mereka adalah Reyna.
```

#### Test Case 2 (Input Valid, Salah Satu Input Merupakan Leluhur Input Lain)
```
Masukkan orang pertama: Clove
Masukkan orang kedua: Cypher
Leluhur mereka adalah Killjoy.
```

#### Test Case 3 (Salah Satu Input Tidak Valid)
```
Masukkan orang pertama: Vyse
Masukkan orang kedua: Asep
Asep siapa bro?!
```

#### Test Case 4 (Kedua Input Tidak Valid)
```
Masukkan orang pertama: Asep
Masukkan orang kedua: Didin
Asep dan Didin siapa bro?!
```

#### Test Case 5 (Salah Satu Input Merupakan Root Tree)
```
Masukkan orang pertama: Yoru
Masukkan orang kedua: Brimstone
Yoru merupakan leluhur paling sepuh jadi gak punya leluhur lagi bos!!!
```
